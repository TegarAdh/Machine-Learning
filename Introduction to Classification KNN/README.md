## About K-Nearest Neighbors (KNN)
KNN is a supervised learning algorithm used for classification and regression tasks. In classification problems, KNN predicts the label of a new data point by identifying the K nearest training samples based on distance metrics such as Euclidean distance.

## Advantages

* Simple and intuitive algorithm
* No explicit training phase
* Effective for small to medium datasets
* Suitable for multiclass classification

## Limitations

* Sensitive to feature scaling
* Slower on large datasets
* Performance depends on choosing optimal K value
